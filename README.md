# PROYECTO-LICEO-CREDITOS

PLATAFORMA PARA EL REGISTRO DE CRÉDITOS EN LICEO UPG / CRISTIAN JORELL LAGUNA GONZÁLEZ

Problemática: 

En base a comentarios de muchos estudiantes de la institución (incluyéndome), se ha registrado una mala experiencia con el registro de créditos extracurriculares
debido a que no es consistente ni preciso, además de que su funcionamiento no resulta eficiente.

Se comenta que muchos cuentan con créditos ausentes o incompletos, que no hay información acerca de a qué pertenecen los créditos o una descripción clara del motivo o razón de la asignación misma,
además de un ausente registro cronológico donde se indique tanto fecha como horario del anexo de los créditos otorgados.

Personalmente el sistema o registro de créditos no es claro ni preciso, y genera una noción ambigua de cuantos crédito se poseen.  


PROBLEMAS:
	1.- No existe una plataforma dedicada
	2.- El estudiante no posee acceso directo o personal
	3.- La información no es precisa
	4.- No existe una interfaz

SOLUCIONES _HERRAMIENTAS:
	1.- Base de datos 
	2.- Sistema de historial / bitácora
	3.- Algoritmos y estructuras elaboradas
	4.- Interfaz gráfica


MEJORAS:
	1.- Plataforma online
	2.- Accesibilidad al público 
	3.- Registro exacto y asegurado de la información
	4.- Interfaz amigable con el usurario e intuitiva


Primer Fase

- Planteamiento del problema
- Idear una solución
- Desarrollo y estructuración de la plataforma

Segunda Fase
- Implementación de herramientas adicionales
- Pulir interfaz y mejorar el aspecto visual
- Agilizar la eficiencia de la plataforma
